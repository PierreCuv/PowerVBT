  GPIO6  (D5)  → I2C SDA MPU6050 (existant)
  GPIO43 (D6)  → I2C SCL MPU6050 (existant)

  │                                      │                         │

  Potentiomètre Bourns 3590S (10 tours, 10kΩ) :
  ┌─────────────────────────────────────────────────────────────────┐
  │                                                                 │
  │   Broche 1 (sens horaire  / CW)  ────────────────── +3V3       │
  │   Broche 2 (curseur / WIPER)     ──── J1 pin 2 (SIG)           │
  │   Broche 3 (sens anti-horaire / CCW) ──────────────── GND      │
  │                                                                 │
  └─────────────────────────────────────────────────────────────────┘

  Filtre RC sur l'entrée ADC :
  ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄
  R1 + C1 forment un filtre passe-bas : f_coupure = 1/(2π × 1kΩ × 100nF) ≈ 1.6 kHz
  → élimine le bruit hautes fréquences de l'ADC de l'ESP32 (connu pour être bruité)
